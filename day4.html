Differences between Comparison Operator and Logical Operator

Comparison Operator

They compare two values and return a Boolean (true or false)

Example:

* == -> equal to (loose Comparison)

* === -> equal value and type (Strict Comparison)

* != -> not equal

* !== -> not equal to value or type

* > -> greater than

* < -> less than

* >= -> greater than or equal to

* <= -> less than or equal to

use case:

5 > 3 // true "a" === "a" //true

Logical Operator

They are used to combine multiple conditions (usually comparisons) and return a boolean.

Examples:

* && -> logical AND (true if both are true)

* || -> Logical OR (true if at least one is true)

* ! -> Logical NOT (reverse a Boolean)

Use Cases:

let age = 20;

if (age > 18 && age < 30){

console.log("you are young!");
