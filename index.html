History of Javascript From ECMAscript (ES1) to (ES6)


Javascript & ECMAscript Overview: Javascript was created by Brendan Eich in 1995 at Netscape. to standardize it accross browsers, it was submitted to ECMA International, leading to the ECMAscript(ES) specification.

ECMAscript1 (1997):first offical version, Basic syntax, types and control structures

ECMAscript2 (1998):Minor changes to align with ISO standards.

ECMAscript3 (1999):Major update, regular expressions, try/catch, swiytch, do-while, delete, typeof,in were introduced

ECMAscript4 (cancelled):proposed major changes,but was abandon due to disagreements (too ambitious, was never released).

ECMAscript5 (2009): huge update after 10 years introduced strict mode, objetcs.keys(), array methods like firEach, map, filter and JASON support.

ECMAscript6 (2015): most important version introduced let, const variables, arrow function ()=>, classes, Template literals, Destructuring, Modules (import / export)
